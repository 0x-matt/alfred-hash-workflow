# alfred-hash-workflow
Using Alfred workflow to get a hash value of input string

## Requirements
---
1. ruby (/usr/bin/ruby)
2. Alfred 4 (Maybe Alfred 3 also work out)

## About
---
Hashes strings with algorithms of ruby md5, sha1, sha256 digest.
